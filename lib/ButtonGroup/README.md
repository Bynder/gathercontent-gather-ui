# ButtonGroup
A group wrapper for buttons.

## Usage

### Props

| Name                  | Type          | Default       | Required | Description                                         |
| --------------------- |-------------- | ------------- | -------- |---------------------------------------------------- |
| children              | node(s)       | n/a           | Yes      | All buttons relevant to the group.  |

```
<ButtonGroup>
  <Button>...</Button>
  <Button>...</Button>
  <Button>...</Button>
</ButtonGroup>
```
